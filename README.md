Who Knows is a prototype from design research project Maternal Machines: Design Speculations about Fantasies of Care, led by Dr Paulina Yurman and funded by Wellcome (https://wellcome.org/research-funding/funding-portfolio/funded-grants/maternal-machines-design-speculations-about). The code was written by Max Park, who is the Research Assistant.
